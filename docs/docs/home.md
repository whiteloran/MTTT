# 魔法少女超越时间

> 该规则仍然在施工中。

BGM：[第一楽章-ulu](https://music.163.com/song?id=29836662&userid=361166853)

本设定试图展现一群能够时间穿梭的魔法少女们，在整个世界史以及多元时间线上追逐自己的愿望的故事。

在游戏机制上，本规则希望构造出简单方便，易于进行游戏的体系。

这主要包括以下几个部分：

* 对网团战斗的加速
* 对战斗单位变化时的战斗长度控制
* 中途加入长团的适应性
* 判定和车卡的简单化
* 规则和流程的简化
* 等等。

除此之外，本规则受到各类TRPG规则和卡牌游戏的致幻油墨攻击，可能在阅读中产生deja vu.
