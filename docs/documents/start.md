#### 魔法少女怎么确定自己在魔女时间轴上的时间？

有一位滴答魔女在时间长河非常早的位置(早到正史派的魔法少女们禁止任何人继续向前探索)。她有一个神奇的摆，这个摆能永远来回运动。每摆一次，就在时间长河的一个特殊频段上产生了一个往复变化的信号，所有身处滴答魔女之后的凡人时间的魔女，只需要收听这个信号，就能确定自己在魔女时间轴上的时点了。
